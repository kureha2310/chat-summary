## 2. 作業中の質問・判断相談（~150件）

確定作業中に判断に迷うケースの共有。中浜さんの食品表示知識が頻繁に活用されている。

### 主要トピック

- **アルコール/酒類の問い合わせ対象判断** — 蒸留酒(スピリッツ)は対象外、リキュール/果実酒/日本酒は対象。ワインは果実酒扱いで対象。蒸留後に香料添加したもの（例: アブソルートバニラ）は対象。
  - [7/29 アルコール原材料の対応](https://app.slack.com/archives/C08EJ36P9C6/p1753752769040539)
  - [10/29 團さんの酒類判断の質問](https://app.slack.com/archives/C08EJ36P9C6/p1761704790474209)

- **精製微量の表示免除** — 豚脂・牛脂などは高度精製でアレルゲン表示不要のケースあり
  - [9/22 豚脂のアレルゲン表示](https://app.slack.com/archives/C08EJ36P9C6/p1758518935450449)

- **トライアルアカウント** — 「問い合わせ代行なし」のアカウントは問い合わせ依頼にせず要確認で判定済
  - [4/11 トライアルアカウントの対応](https://app.slack.com/archives/C08EJ36P9C6/p1744371465863559)

- **かまぼこ等の怪しい表示** — 表示の不備が複数ある場合は要確認で返却
  - [10/29 金目鯛かまぼこ](https://app.slack.com/archives/C08EJ36P9C6/p1761645647651979)

- **「要確認」の使いどころ** — アレルゲンに疑義がある場合のみ使用。表示違反だけでは使わない
  - [11/11 要確認の使用基準](https://app.slack.com/archives/C08EJ36P9C6/p1762842188613039)

## 3. AIタグ・システム誤検知の報告（~80件）

9月以降、尾関さん・團さんの参入で報告件数が急増。

### よくある誤検知パターン

| 誤検知パターン | 例 |
|--------------|---|
| 地名・品種名に反応 | 「りんご和牛」→バラ科、「ごまさば」→ごま |
| 部位名に反応 | 「もも正肉」→桃、「鶏肉（もも）」→桃 |
| 否定表現に反応 | 「小麦不使用しょうゆ」→小麦 |
| ミルク系に反応 | 「ココナッツミルク」→乳 |
| 魚卵系に反応 | 「えびの卵」→卵、「助宗たら卵」→卵、「カラフトししゃも卵」→卵 |
| 動物脂に反応 | 「ラード」→豚肉 |
| 果物名に反応 | 「山もも」→桃 |
| 酒名に反応 | 「マルサラマスト」→さけ |
